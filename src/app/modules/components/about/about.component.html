About this Project


The purpose of this project is for me to learn Angular. I have previously made the same project using
PHP, HTML, CSS so the database is the same. I implemented the Backend using Java Spring Boot
and the Frontend using Angular. More specifically, the technologies that are used are :


Angular

Angular is a popular open-source web application framework developed by Google. It simplifies the development of dynamic, single-page web applications by providing a structured framework for building client-side applications using TypeScript.

Spring Boot

Spring Boot is an open-source Java framework that simplifies and accelerates the development of production-ready, stand-alone, and web-based applications by providing a convention-over-configuration approach and a range of built-in features.

MDBoostrap

MDBootstrap (or MDB) is a popular front-end framework for building responsive and modern web applications using HTML, CSS, and JavaScript, with a focus on Material Design principles and components.
